## 👤 About

This is my personal CV/resume website showcasing my education, skills, and experience. Built with clean HTML and CSS for optimal performance and accessibility.

## ✨ Features

- 📱 **Fully Responsive** - Looks great on all devices
- 🎨 **Clean Design** - Professional and modern layout
- ⚡ **Fast Loading** - Pure HTML/CSS, no frameworks
- 🖨️ **Print-Friendly** - Optimized for PDF export
- 🔗 **Social Links** - Connect with me on various platforms
- 📧 **Contact Form** - Easy way to get in touch

## 🛠️ Tech Stack

- **Structure:** HTML5 (Semantic markup)
- **Styling:** CSS3 (Flexbox, Grid, Custom Properties)
- **Icons:** Font Awesome
- **Fonts:** Google Fonts
- **Hosting:** GitHub Pages

## 📋 Sections

- **Header** - Name, title, contact info
- **About** - Brief introduction
- **Education** - Academic background
- **Skills** - Technical and soft skills
- **Experience** - Work history
- **Projects** - Portfolio highlights
- **Contact** - Contact information and form

## 🔧 Customization

1. Edit `index.html` to update content
2. Modify `style.css` to change styling
3. Replace images in `assets/` folder
4. Update links in the contact section

## 📱 Responsive Breakpoints

- **Desktop:** 1024px+
- **Tablet:** 768px - 1023px
- **Mobile:** < 768px
